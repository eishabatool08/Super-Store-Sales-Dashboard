# 📊 Superstore Performance Dashboard (Microsoft Excel)

An interactive and dynamic business intelligence dashboard built entirely in Microsoft Excel to analyze retail superstore data across multiple years and regions.

## 🔑 Key Features & Technical Skills Applied
* **Dynamic Slicers:** Integrated interactive timelines/slicers for `Years (2011-2014)` and `Regions` to filter data dynamically.
* **Advanced Data Modeling:** Cleaned and structured raw transactional data using **Pivot Tables** and **Excel Formulas** across multiple sheets (`Detail1`, `Detail2`, `Calculation`).
* **KPI Metrics:** Tracked critical business health signs: *Total Sales*, *Total Profit*, and *Profit Margin %*.
* **Diverse Visualizations:** 
  - **Trend Analysis:** Line chart representing *Total Sales of Year*.
  - **Geographic Spread:** Pie chart showing *Profit of Regions*.
  - **Category Breakdown:** Focused line and horizontal bar charts for *Categorywise Sales* and *Sub-category of Sum of Sales*.
  - **Top Performers:** Column chart highlighting *City wise Sum of Sales* (e.g., New York City, Los Angeles).

---

## 📂 File Structure
* `Sales Dashboard` - Main presentation layer for stakeholders.
* `Calculation` & `Detail Sheets` - Back-end data transformation and calculations.
* `Data Table` - Cleaned master raw dataset.
